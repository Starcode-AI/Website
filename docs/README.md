# Starcode BOM Link Hub

Fertige statische Website für GitHub Pages. Kein Build und keine Installation nötig.

## Veröffentlichung

1. Den **Inhalt dieses Ordners** in dein GitHub-Repository hochladen.
2. Unter **Settings → Pages** den Branch `main` und `/ (root)` wählen.
3. Warten, bis GitHub die fertige URL anzeigt.

## Links bearbeiten

Alle Links liegen in `assets/js/config.js`. `url: null` bedeutet: Die Karte wird angezeigt, öffnet aber noch keinen Link. Dort müssen noch der echte Discord-Einladungslink, PayPal.me-Link und die Bitcoin-Adresse bzw. Payment-URL ergänzt werden. Bitte außerdem prüfen, ob das AI-System-Repository wirklich `https://github.com/hackerIP54/AI-system` heißt.

Texte/Übersetzungen: `assets/js/translations.js`  
Farben: `assets/css/tokens.css`  
Logo: `assets/img/logo.svg`
